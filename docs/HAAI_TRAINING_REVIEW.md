# HAAI Training Review

HAAI creates evidence that can be reviewed by APV, humans, and future AI assistants.

Training review categories:
- clear success
- clear failure
- tool misuse
- missing context
- hallucination candidate
- corrected output
- retry trail
- final accepted answer

HAAI Core does not decide these labels.

Core provides:
- run records
- event streams
- hashes
- packets
- diffs
- links
- receipts

APV and packs consume verified evidence and classify it.
