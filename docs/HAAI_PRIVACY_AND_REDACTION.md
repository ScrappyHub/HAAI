# HAAI Privacy and Redaction

HAAI records powerful AI evidence, so privacy is mandatory.

Default rules:
- local-first
- no cloud upload by default
- explicit user action for export
- domain allowlist for browser capture
- pause and stop capture controls
- redact secrets before packet build

Sensitive material classes:
- API keys
- access tokens
- private keys
- passwords
- session cookies
- personal identifiers
- private source files
- proprietary prompts or datasets

Core records evidence.
Policy packs can label or redact, but Core must preserve verifiable boundaries.
